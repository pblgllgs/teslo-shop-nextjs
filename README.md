# Next.js teslo-store

para correr localmente se necesita la bd

```
docker-compose up -d
```

# Mongo url local:

```
mongodb://localhost:27017/teslodb
```

## instalar dependiencias

```
npm install

pnpm install

yarn
```

## Configurar variables de entorno

renombrar el archivo __.env.template__ a __.env__
